### Usage

```
usage: mason_image [-h] [--in IN_IMAGE] [--out OUT_IMAGE]

optional arguments:
  -h, --help         show this help message and exit
  --in IN_IMAGE      path of image to mess up
  --out OUT_IMAGE    path of output image
```


### Install

```
$ pip install mason_image
```
